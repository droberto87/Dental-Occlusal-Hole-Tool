# Dental Occlusal Hole Tool - Roadmap

## ✅ v1.0 - Stable Release (Current)
- High-performance realtime visualization (60fps).
- Shader-based hole transparency with internal wall simulation.
- Single-click (point-and-shoot) placement.
- Marker-based interaction (drag to move, scroll to resize).
- Card-based sidebar for channel management.
- Robust BVH-accelerated boolean export.
- Undo/Redo history system.

## 🚀 v1.1 - Precision & UI Polishing (Next Steps)
- [ ] **Angle Snapping**: Option to snap holes to specific axes (X, Y, Z) or surface normals.
- [ ] **Diameter Presets**: Save commonly used hole diameters (e.g., 2.5mm, 3.2mm).
- [ ] **Multi-selection**: Edit diameter of multiple holes at once.
- [ ] **Visual Feedback**: Improved distance rulers between multiple holes.

## 🌟 v2.0 - Advanced Features (The "Idea Cannon")
- [ ] **Collision Detection**: Warning if two holes are too close to each other.
- [ ] **Drill Bit Simulation**: Animated drill entering the model to check clearance.
- [ ] **Project Persistence**: Save/Load projects in a custom format to keep holes editable later.
- [ ] **Batch Processing**: Apply the same hole pattern to multiple STL models automatically.
